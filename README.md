Tema: Sistema de Funcionários (CLT e PJ)

O que esse projeto demonstra ->

- Abstração: classe abstrata Funcionario
- Encapsulamento: atributos private + getters/setters
- Herança: FuncionarioCLT e FuncionarioPJ herdam de Funcionario
- Polimorfismo: método calcularPagamento() sobrescrito em cada tipo

Estrutura ->
 > src/
       - Funcionario.java
       - FuncionarioCLT.java
       - FuncionarioPJ.java
       - Main.java
